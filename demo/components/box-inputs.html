Box Input Components

Interactive box input controls following EchoPark Design System tokens

Box Input XS

The box input component provides selectable options in a clean, accessible format. Built with design tokens for consistency.

States Overview

Default
Hover
Selected
Focus Selected
Design Tokens Used:
Colors: --color-base-white, --color-neutral-95, --color-base-primary-50, --color-feedback-accessibility-80
Spacing: --spacing-inset-squish-nano
Border Radius: --border-radius-md
Typography: --typography-font-family-base, --typography-font-size-base

Interactive Demo

Try clicking the options below:

Specifications

Property Value Token
Height 48px Fixed (XS size)
Min Width 175px Fixed
Padding 8px 16px --spacing-inset-squish-nano
Border Radius 8px --border-radius-md
Font Size 16px --typography-font-size-base
Line Height 24px (150%) Fixed
Default Border 1px solid --color-neutral-90
Selected Border 2px solid --color-base-primary-50
Focus Border 3px solid --color-feedback-accessibility-80

Usage Code

<!-- HTML --> <button class="box-input" aria-pressed="false"> Label </button> <!-- React Component --> import { BoxInput } from '@echopark/design-system'; <BoxInput size="xs" selected={false} onClick={handleClick} aria-label="Select option" > Label </BoxInput>
